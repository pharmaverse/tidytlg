# Removes all borders from the table

Removes all borders from the table

## Usage

``` r
no_borders(ht, matrix = NULL)
```

## Arguments

- ht:

  `huxtable` object.

- matrix:

  `matrix` of bottom borders. Ignored. Included for the sake of
  compatibility with the interface of all border mutating functions.
