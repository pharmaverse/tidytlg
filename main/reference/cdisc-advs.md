# `ADVS` data created from subsetting the `CDISC` `ADVS` dataset

`ADVS` data created from subsetting the `CDISC` `ADVS` dataset

## Usage

``` r
cdisc_advs
```

## Format

A data frame with 1938 rows and 35 variables:

- `STUDYID`:

  Study Identifier

- `SITEID`:

  Study Site Identifier

- `USUBJID`:

  Unique Subject Identifier

- `AGE`:

  Age

- `AGEGR1`:

  Pooled Age Group 1

- `AGEGR1N`:

  Pooled Age Group 1 (N)

- `RACE`:

  Race

- `RACEN`:

  Race (N)

- `SEX`:

  Sex

- `SAFFL`:

  Safety Population Flag

- `TRTSDT`:

  Date of First Exposure to Treatment

- `TRTEDT`:

  Date of Last Exposure to Treatment

- `TRTP`:

  Planned Treatment

- `TRTPN`:

  Planned Treatment (N)

- `TRTA`:

  Actual Treatment

- `TRTAN`:

  Actual Treatment (N)

- `PARAMCD`:

  Parameter Code

- `PARAM`:

  Parameter

- `PARAMN`:

  Parameter (N)

- `ADT`:

  Analysis Date

- `ADY`:

  Analysis Relative Day

- `ATPTN`:

  Analysis Timepoint (N)

- `ATPT`:

  Analysis Timepoint

- `AVISIT`:

  Analysis Visit

- `AVISITN`:

  Analysis Visit (N)

- `AVAL`:

  Analysis Value

- `BASE`:

  Baseline Value

- `BASETYPE`:

  Baseline Value

- `CHG`:

  Change from Baseline

- `PCHG`:

  Percent Change from Baseline

- `VISITNUM`:

  Visit Number

- `VISIT`:

  Visit Name

- `VSSEQ`:

  Sequence Number

- `ANL01FL`:

  Analysis 01 - Special Interest Flag

- `ABLFL`:

  Baseline Record Flag

## Source

CDISC SDTM/ADAM Pilot Project.
